- gitbash down
- git 로그인 후 repository 생성 
- 로컬 피씨안 공유폴더에서 bash 실행

- $ git status - 현재 폴더 상황 확인 (생성한 파일은 빨간색으로 보임) 
- $ git add 파일명 또는 $ git add * (스테이징 영역으로 보냄//초록색으로 보임)
- $ git commit -m "description" (커밋 영역) 
	//커밋을 해주시면 스테이징영역에 있던 모든 파일들이 커밋됩니다.
	//커밋이 되면 기록에 변경사항, 메모들이 남습니다.
	//나중에 커밋한 내용을 모두 확인할 수 도 있고 특정 커밋 시점으로 코드를 불러올 수 있습니다.
- $ git push origin master(저장소로 보내버림)
